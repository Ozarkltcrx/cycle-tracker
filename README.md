# Command Center

Missouri LTC pharmacy command center for Turner × Thorpe operations.

## Features

- **Dashboard** — Missouri LTC operator view with morning risk board, facility watchlist, and workflow priorities
- **Projects** — Track active workstreams, create new ones, view status
- **Messages** — Communication log between Turner and Thorpe
- **Quick Notes** — Fast capture for ideas and reminders

## Projects Tracked

Projects are auto-discovered from `~/.openclaw/workspace/projects/` — any folder with a `README.md` shows up here.

## Run

```bash
streamlit run app.py
```

Or use the desktop app on your Desktop.
